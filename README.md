# Salero Pakan

Deskripsi singkat project

## Fitur
- Login Admin & User
- CRUD Supplier
- CRUD Produk
- Laporan
- Dashboard
- Session
- JOIN Database

## Teknologi
- PHP Native
- MySQL
- Bootstrap 5
- Laragon

## Cara Menjalankan
1. Clone repository
2. Import database
3. Jalankan Laragon
4. Buka localhost
5. Login sebagai Admin atau User

## Instalasi
1. Clone repository.
2. Pindahkan project ke folder `laragon/www/`.
3. Import database `database/salero_pakan.sql` melalui phpMyAdmin.
4. Jalankan Apache dan MySQL di Laragon.
5. Buka:

http://localhost/salero-pakan
